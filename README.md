# Game of Set

## About

The Game of Set is a card game where players identify sets of three cards out of a deck of twelve that share specific properties.
Each card has four attributes:
- **Number**: 1, 2, or 3
- **Shape**: Diamond, Squiggle, or Oval
- **Color**: Red, Green, or Purple
- **Shading**: Solid, Shaded, or Empty

A valid set consists of three cards where each attribute is either all the same or all different across the three cards.

This application currently uses a exhaustive search algorithm to generate solutions.

It is publicly deployed using Vercel, which can be viewed with this link: https://game-of-set-nine.vercel.app/.

## Running the Dev Server

To start the development server, run:

```bash
npm run dev
```

The server will start locally and be accessible in your browser. Changes to the code will automatically reload.

## Getting Started

1. Clone or navigate to this repository
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
